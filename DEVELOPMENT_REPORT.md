## 核心特性

### 1. 多Agent协作
- **辩论机制**：辩护者vs批评者，仲裁者综合判断
- **思考帽模型**：6个维度全面分析
- **知识共享**：热知识库+冷知识库

### 2. 智能推理
- **消息面筛选**：基于事件简报初筛
- **技术面分析**：MA、MACD等指标
- **多轮对抗**：发现投资逻辑漏洞

### 3. 持续学习
- **复盘机制**：对比推荐与实际结果
- **经验提取**：自动总结可复用经验
- **知识沉淀**：高置信度经验沉淀到冷知识库

### 4. 灵活配置
- **多LLM支持**：OpenAI、Claude、国内模型
- **自定义API**：支持任意兼容OpenAI格式的API
- **模块化设计**：易于扩展和定制

## 使用方法

### 初始化
```bash
pip install -r requirements.txt
cp .env.example .env  # 填入API密钥
python main.py init
```

### 运行工作流
```bash
python main.py observe  # 观察市场
python main.py reason   # 获取推荐
python main.py review   # 复盘分析
```
